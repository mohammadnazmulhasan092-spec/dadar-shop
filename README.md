# Dadar Shop - Your Trusted Online Marketplace

A modern, responsive eCommerce website built with HTML, CSS, and JavaScript.

## Pages Included

- Homepage - Hero slider, flash sale, categories, featured products
- Categories - Electronics, Fashion, Beauty, Home & Living, Groceries, Books, Sports, Toys
- Product Detail - Gallery, options, add to cart
- Shopping Cart - Cart items, order summary
- Checkout - Shipping, payment, order placement
- Authentication - Login, Register, Forgot Password
- Support - Contact Us, About Us, FAQ, Help Center

## Features

- Responsive mobile-first design
- Flash sale countdown timer
- Product filtering and sorting
- Shopping cart functionality
- Multi-step checkout
- Mobile bottom navigation

## How to Use

1. Clone or download the repository
2. Open index.html in any web browser
3. Navigate through the website

## Tech Stack

- HTML5
- CSS3 (with CSS Variables)
- Vanilla JavaScript
- Pexels stock images
